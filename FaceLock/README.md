# FaceLock – Facial Authentication for Windows

## Overview

FaceLock is a privacy-focused biometric authentication system designed to automatically lock and unlock a Windows workstation using facial recognition technology.

The project leverages Computer Vision and Deep Learning techniques to provide a secure and seamless user experience. When the authorized user leaves the workstation, the system automatically locks the Windows session. Upon returning, the user is authenticated through a webcam and the session is unlocked if a valid facial match is detected.

Inspired by privacy-by-design principles, FaceLock processes biometric data locally without relying on cloud services, ensuring that sensitive information remains under the user's control.

## Objectives

* Develop an automated Windows session lock/unlock system.
* Implement facial recognition using Deep Learning.
* Improve workstation security through passive authentication.
* Protect biometric data using privacy-preserving techniques.
* Provide a seamless and user-friendly authentication experience.

## Key Features

### Real-Time Face Detection

The system continuously monitors webcam input and detects the presence of a user in real time.

### Facial Recognition

Authorized users are identified using facial embeddings generated by pre-trained Deep Learning models.

### Automatic Session Locking

When the user leaves the workstation:

* User absence is detected.
* The Windows session is automatically locked.
* Unauthorized access risks are reduced.

### Automatic Session Unlocking

When the authorized user returns:

* Facial recognition is performed.
* The captured face is compared with stored biometric templates.
* The session is automatically unlocked upon successful authentication.

### Local Biometric Database

Instead of storing raw facial images, the system stores encrypted biometric templates (embeddings) locally.

### Privacy-by-Design

The solution incorporates privacy principles including:

* Local processing only
* No cloud storage
* No transmission of biometric data
* Encrypted biometric templates
* Reduced privacy risks
